`mille` は、クリーンアーキテクチャ・オニオンアーキテクチャ・ヘキサゴナルアーキテクチャなど、**レイヤードアーキテクチャの依存ルールを静的解析する CLI ツール**です。

## チェック機能

| チェック | Rust | Go | TypeScript | JavaScript | Python | Java | Kotlin | PHP |
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| レイヤー依存ルール (`dependency_mode`) | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 外部ライブラリルール (`external_mode`) | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| DI メソッド呼び出しルール (`allow_call_patterns`) | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
